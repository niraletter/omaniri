# Omaniri

Omaniri is a beautiful, modern & opinionated Arch Linux distribution with [niri](https://github.com/YaLTeR/niri) scrollable-tiling Wayland compositor by N1R4.

### Install

```bash
curl -fsSL https://niraletter.github.io/omaniri | bash
```

See [Manual Installation](docs/manual-installation.md) if you can't use the ISO.
